## 🧭 devPort - 글로벌 개발 트렌드 큐레이션 서비스


오픈소스 프로젝트와 급상승 중인 최신 기술 블로그를 한 곳에서, 한국어 요약과 함께 확인할 수 있는 서비스입니다.
HashNode, Medium, Reddit 등 해외 개발 커뮤니티 곳곳에 흩어진 정보를 모아 한국 개발자를 위한 맞춤 요약으로 전달합니다.

영어 때문에 놓쳤던 트렌드, 시간이 없어서 놓쳤던 뉴스 이제 5분이면 해외 개발 트렌드를 따라갈 수 있습니다.
